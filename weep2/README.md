# 🐕 الصاحة الفاخرة للكلاب - Luxury Dog Resort

موقع إلكتروني حديث ومتكامل لصاحة كلاب فاخرة مع 8 صفحات متكاملة، تصميم عصري، وميزات إمكانية وصول متقدمة.

## ✨ المميزات الرئيسية

### 📱 **التصميم المتجاوب**
- متوافق تماماً مع جميع الأجهزة (سطح المكتب، الجوال، التابلت)
- شريط جانبي ثابت يتحول إلى قائمة جوال
- تصميم عصري بألوان بيج وبني فاخر

### 🌍 **دعم متعدد اللغات**
- **العربية** (الافتراضية) - اتجاه RTL
- **English** - اتجاه LTR  
- **中文** (الصينية) - اتجاه LTR
- تغيير تلقائي لاتجاه الصفحة حسب اللغة

### ♿ **إمكانية الوصول**
- أزرار تكبير وتصغير الخط (A+ / A-)
- تباين ألوان واضح
- دعم لوحة المفاتيح
- نصوص بديلة للصور
- هيكل HTML دلالي صحيح

### 🧭 **التنقل السهل**
- شريط جانبي ثابت مع 8 روابط
- قائمة جوال للأجهزة المحمولة
- تفعيل/إلغاء تفعيل الشريط الجانبي
- تأثيرات hover احترافية

## 📑 **صفحات الموقع (8 صفحات)**

### 1️⃣ **الصفحة الرئيسية** (`index.html`)
- قسم ترحيبي بصورة كبيرة
- عرض مميزات الصاحة
- نبذة عن الخدمات
- زر دعوة للعمل (CTA)

### 2️⃣ **من نحن** (`about.html`)
- تعريف بالشركة
- الرؤية والرسالة
- خبرة الفريق
- التزام براحة الحيوانات
- إحصائيات وإنجازات

### 3️⃣ **المرافق** (`facilities.html`)
- غرف الإقامة المريحة
- التلفزيون والترفيه
- المراعي الخارجية الآمنة
- حظيرة اللعب الداخلية
- مسار الرشاقة
- خدمات بيطرية 24 ساعة
- خدمة التوصيل

### 4️⃣ **الحزم** (`packages.html`)
- **الباقة اليومية** (50 ريال/اليوم)
- **الباقة الكلاسيكية** (80 ريال/اليوم)
- **الباقة المميزة** (120 ريال/اليوم)
- باقات مخصصة للإقامات الطويلة
- باقات للكلاب ذات الاحتياجات الخاصة
- جدول مقارنة بين الباقات

### 5️⃣ **العناية والتجميل** (`grooming.html`)
- الاستحمام الكامل
- جلسة القص والتشذيب
- التنظيف والعناية
- جلسة الاسترخاء
- عناية خاصة
- الباقة الكاملة المميزة

### 6️⃣ **الرعاية الصحية** (`healthcare.html`)
- خدمات طبية طارئة 24 ساعة
- الفحوصات الدورية
- التطعيمات واللقاحات
- صحة الفم والأسنان
- المختبر والتحاليل
- الرعاية الدوائية
- التغذية الصحية

### 7️⃣ **الأسئلة الشائعة** (`faq.html`)
- نظام بحث متقدم
- تصنيف الأسئلة (حجز، خدمات، صحة، دفع)
- إجابات تفصيلية
- قسم للتواصل المباشر

### 8️⃣ **تواصل معنا** (`contact.html`)
- معلومات الاتصال الكاملة
- نموذج تواصل تفاعلي
- نموذج حجز متكامل
- خريطة الموقع
- روابط وسائل التواصل الاجتماعي
- رقم طوارئ
