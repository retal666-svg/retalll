# 🔎 البحث الذكي (ظاهر وواضح)

✔ يظهر بشكل واضح أعلى الشريط الجانبي
✔ عند الكتابة تظهر اقتراحات مباشرة أسفله
✔ يعمل أثناء الكتابة (Live Search)
✔ ينقل المستخدم مباشرة إلى الصفحة أو القسم عند الضغط

### مثال:

إذا كتب المستخدم:

* "ر" → تظهر: الرعاية الصحية – الرئيسية
* "ح" → تظهر: الحزم – حظيرة لعب داخلية – حفر حفرة في الرمل
* "م" → تظهر: المرافق – مسار الرشاقة – مراقبة الحيوانات

الميزة تساعد المستخدم للوصول السريع لأي خدمة بسهولة.
